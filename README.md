Um jogo de adivinhação desenvolvido com HTML, CSS e JavaScript puro, onde o jogador tenta descobrir um número secreto gerado aleatoriamente entre 1 e 50.

🕹️ Como jogar

O jogo sorteia um número secreto entre 1 e 50
Digite seu palpite no campo de input e clique em "Chutar"
O jogo informa se o número secreto é maior ou menor que o seu chute
Continue tentando até acertar!
Ao acertar, o jogo exibe quantas tentativas foram necessárias e o botão "Novo jogo" é habilitado

✨ Funcionalidades

Geração de números aleatórios sem repetição (o mesmo número não é sorteado duas vezes consecutivas até que todos os números do intervalo tenham sido utilizados)
Feedback por voz com a biblioteca ResponsiveVoice (voz feminina em português brasileiro)
Contador de tentativas com concordância gramatical correta ("1 tentativa" / "2 tentativas")
Botão de reinício desabilitado durante o jogo e habilitado apenas após o acerto
